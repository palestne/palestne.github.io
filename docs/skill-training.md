# Skill Training Methods

This page will document the skill training methods employed for each skill.

______

**FM** 

|  1 - 50 | Chopping and burning logs (normal, oaks, willows) | 
|  50 - 75 | Wintertodt | 
|  75 - 99 | Not required | 

**WC**

|  1 - 50 | Test | 

**Agility**

|  1 - 40 | Highest level roof top available (skip Al-Kharid)|
|  40 - 52 | Brimhaven (preferably in one go, as it requires GP to enter, should be ~5 hours) |
|  52 - 76 (1.45m exp) | Wilderness agility | 
| 76 - 80 | Quests | 
| 80 - 99 | Not required |
